# Blog Pribadi - Portofolio Saya

## 📝 Tentang Website

Website blog pribadi ini merupakan portofolio digital yang menampilkan perjalanan pendidikan, bakat, pengalaman, dan informasi kontak saya. Dibangun dengan teknologi web modern untuk memberikan pengalaman browsing yang responsif dan menarik.

## ✨ Fitur Utama

- **🏠 Beranda**: Halaman sambutan dengan navigasi smooth scroll
- **📚 Pendidikan**: Riwayat pendidikan dari SMK hingga kuliah
- **⭐ Bakat & Skill**: Kemampuan di bidang web development, UI/UX, dan programming
- **💼 Pengalaman**: Timeline pengalaman kerja dan magang
- **📧 Kontak**: Formulir kontak dan informasi sosial media

## 🛠️ Teknologi yang Digunakan

- **HTML5**: Struktur dan markup
- **CSS3**: Styling dan animasi
- **JavaScript**: Interaktivitas dan fungsionalitas

## 🚀 Akses Website

Kunjungi website live di: [https://cre-web-v2-vc-blog-pribadi.vercel.app/](https://cre-web-v2-vc-blog-pribadi.vercel.app/)

## 📱 Responsive Design

Website ini dirancang untuk dapat diakses dengan baik di berbagai perangkat:
- Desktop
- Tablet
- Mobile
